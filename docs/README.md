# Docs

This folder stores long-lived project documents.

Suggested organization:

- `00_project_guidance/`: project rules, workflows, and go/no-go criteria.
- `01_literature/`: paper notes and reading maps.
- `02_theory/`: derivations, equations, and unresolved theory questions.
- `03_meetings/`: meeting notes and advisor discussions.
- `04_reports/`: internal reports, slides outlines, and paper-planning notes.
- `05_reference_projects/`: notes on external projects and architecture patterns.
