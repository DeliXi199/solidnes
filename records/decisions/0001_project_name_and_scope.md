# Decision 0001: Project Name And Initial Scope

Date: 2026-05-21

## Decision

Use `SolidNES` as the project name.

Full name:

```text
Neural Quantum States for Excited States of Periodic Solids
```

## Initial Scope

The first concrete target is a carbon-diamond periodic-solid benchmark that can
support later excited-state NQS workflow development.

## Rationale

This name is short, specific enough to signal the research direction, and broad
enough to include future systems such as MoS2, NiO, and defect
embedding without renaming the project.

The initial scope is intentionally narrow because the project needs an early
go/no-go signal before committing to expensive strong-correlation or defect
calculations.
