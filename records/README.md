# Records

This folder stores chronological project memory.

Use:

- `progress/` for dated status notes.
- `decisions/` for explicit research and engineering decisions.
- `logs/` for human-readable run notes.
- `run_index.md` for numbered task-bundle tracking.

For new runs, raw machine logs and checkpoints belong in the relevant
`tasks/.../NNNN_short_slug/` bundle, not here. Legacy raw outputs may still live
under top-level `results/`, `outputs/`, and `logs/`.
